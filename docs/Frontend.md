# Arquitectura del Frontend

## HomePage

La HomePage no contiene toda la interfaz.

Su función es únicamente unir componentes.

Estructura:

```text
HomePage
│
├── Hero
├── Features
├── UpcomingEvents
├── CallToAction
└── Footer
```

## Responsabilidad de cada componente

### Hero

Presenta el mensaje principal de la iglesia.

### Features

Muestra las principales áreas del portal.

### UpcomingEvents

Lista los próximos eventos.

### CallToAction

Invita al usuario a registrarse o donar.

### Footer

Información de contacto, redes sociales y enlaces.
